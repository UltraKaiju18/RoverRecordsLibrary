# Rover Records Library 🎵

A voice-activated music genre lookup app built as a PWA (Progressive Web App) for Android.

## Features
- 🎤 Speak an artist name → hear and see their genre instantly
- ⌨️ Type fallback search for when voice isn't practical
- 💿 Browse your full library with searchable, color-coded genre tags
- 📂 Upload your Discogs CSV export directly in the app
- 📱 Installable to Android home screen — works offline

## CSV Format
Your Discogs export must include these columns:
- `RoverRecord Artist`
- `RoverRecord Bent Genre`
